# 🎓 Student Dropout Prediction System

## 📌 Overview

This project is an AI-powered web application that predicts the risk of student dropout based on multiple academic and personal factors.

It helps educators and institutions identify at-risk students early and take preventive actions.

---

## 🚀 Features

* 📊 Predicts student dropout risk (Low / Medium / High)
* 📈 Interactive dashboard using Streamlit
* 🧠 Machine Learning-based prediction model
* 📉 Visual insights and analytics
* 🔄 What-if analysis (simulate different scenarios)

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Plotly

# ▶️ How to Run the Project

# 1. Clone the repository

```
git clone https://github.com/Aditykumari3038/dropout-student-name.git
cd dropout-student-name
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

# 3. Run the app

```
streamlit run app.py
```

 # 📊 How It Works

* User inputs student data (attendance, marks, etc.)
* Data is processed and passed to ML model
* Model predicts dropout probability
* Results are visualized in dashboard
